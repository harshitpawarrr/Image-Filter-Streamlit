# Image Filter App using Streamlit

This is a simple web-based image filter application built using **Python and Streamlit**.  
The app allows users to upload an image and apply basic filters such as grayscale, contour, and blur.

This project was created to practice Python, image processing, and Streamlit basics.

---

## Features
- Upload image (JPG, JPEG, PNG)
- Apply grayscale filter
- Apply contour filter
- Apply blur filter
- Simple and interactive UI

---

## Technologies Used
- Python
- Streamlit
- Pillow (PIL)

## How to Run the Project

### Step 1: Install dependencies
```
pip install -r requirements.txt
```

### Step 2: Run the app
```
streamlit run app.py
```

---

## Output
After running the command, a browser window will open where you can upload an image and apply different filters.
